# 1. 需要使用sudo 方式启动 start-dfs.sh

# 理由:

  1. hadoop 会有创建/删除文件或文件夹等操作,而这些操作,一般需要超级管理员来做,而webHDFS作为API, 不具备手动提权的能力,所以,需要使用sudo方式启动

# 步骤

  1. 改写 ssh的配置文件,使得ssh允许以root方式启动

    ssh的配置文件: /etc/ssh/sshd_config

    UseDNS no
AddressFamily inet
SyslogFacility AUTHPRIV
PermitRootLogin yes
