# 讲讲前后端鉴权的问题

# 跨域身份验证:

  1. 用户向服务器发送用户名和密码。
  2. 验证服务器后，相关数据（如用户角色，登录时间等）将保存在当前会话中
  3. 服务器向用户返回 session_id，session 信息都会写入到用户的 Cookie
  4. 用户的每个后续请求都将通过在 Cookie 中取出 session_id 传给服务器
  5. 服务器收到 session_id 并对比之前保存的数据，确认用户的身份

# JWT

JWT 的原则是在服务器身份验证之后，将生成一个 JSON 对象并将其发送回用户

# JWT 在 nodejs 中实现

需要使用到两个包:

  1. koa-jwt
  2. jsonwebtoken

jsonwebToken 的 github 地址 (opens new window)

# koa-jwt 的配置:

app.use(
  jwt({
    secret,
  }).unless({
    path: [/\/users\/register/, /\/users\/login/],
  })
);

let token = jwtToken.sign(
  {
    data: userOne,
    exp: config.exp(),
  },
  config.secret
);
ctx.body = {
  code: 2000,
  token: token,
