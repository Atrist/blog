# <token>

<token>必须要使用 ``框起来,不然会导致,渲染不通过

# 缘由

markdown 最终会渲染成 html 文档,此时 <token>会变渲染成一个没有结束标签的 html 标签,导致后面的内容全部都不能被正常的渲染
