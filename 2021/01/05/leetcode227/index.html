# 基本计算器 II

实现一个基本的计算器来计算一个简单的字符串表达式的值。

字符串表达式仅包含非负整数，+， - ，*，/ 四种运算符和空格。 整数除法仅保留整数部分。

# 示例 1:

输入: "3+2*2"
输出: 7
1
2

# 示例 2:

输入: " 3/2 "
输出: 1
1
2

# 示例 3:

输入: " 3+5 / 2 "
输出: 5
1
2

# 说明：

  • 你可以假设所给定的表达式都是有效的。
  • 请不要使用内置的库函数 eval。

# 解法

# 思路

  • 从左向右遍历字符串，符号标识f，初始+
  • 空格，忽视。数字，当字符串拼接。非数字，根据f运算
  • +和-入栈，*和/和栈第一位运算，结果入栈
  • 返回栈的累加和

# 代码

/**
 * @param {string} s
 * @return {number}
 */
var calculate = function (s) {
  const q = [];
  let n = "";
  let f = "+";
  for (let i = 0; i < s.length || n; i++) {
    if (s[i] === " ") continue;
    if (/\D/.test(s[i])) {
      switch (f) {
        case "+":
          q.push(n);
          break;
        case "-":
          q.push(-n);
          break;
        case "*":
          q.push(q.pop() * n);
          break;
        case "/":
          q.push((q.pop() / n) | 0);
      }
      f = s[i];
      n = "";
    } else n += s[i];
  }
  return q.reduce((p, v) => p + (v | 0), 0);
};
